Bootcamp 2020: Brick Breaker Game

My first programming project, concluding the 2 week Programming Bootcamp at City(2020). This Processing project demonstrates a comprehensive understanding of the use of classes, loops, and arrays. I have created a Brick Breaker game which consists of three main parts the paddle, ball, and bricks. Using classes these objects were made with greater needs than any other method. An example of where this is evident is with the bricks, using an array I was able to create multiple bricks witht the same attributes in a grid format. The ball will bounce whenever it hits the walls on the side or the top, but will show a game over message once it hits the bottom. When a brick is hit by the ball its collision value is set to true and will then be removed.
